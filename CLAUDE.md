# Expense Tracker

A simple expense tracking web app built with Next.js 15 and Tailwind CSS.

## Stack
- Next.js 15 (App Router) + React 19
- Tailwind CSS for styling
- In-memory storage (`lib/storage.js`) — data resets on server restart

## Running Locally

    npm install
    npm run dev

Open http://localhost:3000
